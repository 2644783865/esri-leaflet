npm

Esri Leaflet is also available on npm and can be installed with the following command.

npm install esri-leaflet --save

Bower

Esri Leaflet is available on Bower as well, and can be installed with the following command.

bower install esri-leaflet

Still using Leaflet 0.7.x?

Version 1.0.3 of esri-leaflet is the one for you.
